Usage Instructions:

  • Success: Use for successful operations (saves, generations)
  • Error: Use for failed operations or validation errors
  • Warning: Use for caution messages or partial success
  • Info: Use for general information or status updates

Implementation Examples:

// Simple usage
showNotification('Topic saved successfully!', 'success');

// With custom duration
showNotification('Processing...', 'info', 2000);

// Using convenience methods
SimpleNotifications.success('Operation completed!');
SimpleNotifications.error('Something went wrong');
